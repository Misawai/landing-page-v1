## 警告制度 *(版本 v1.0, 20210525)*
如玩家有不當行為時，管理員會透過 **聊天室、顯示標題** 等方式提醒：

- 提醒後立即改善者無罰則，列入觀察名單中，若再有不當行為將予以警告一次。
- 提醒後不聽者，記警告一次。
- 提醒多次後無回應（掛機）者，一概先移除白名單並踢出遊戲。待聯繫到玩家、處理完畢後，再決議是否重新加入。
- 警告達兩次以上者，暫時取消進入伺服器資格。
- 管理員得視情況調整處罰輕重。

*管理員保有更改與最終解釋權利。*
